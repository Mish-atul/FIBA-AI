# FIBA AI

Find-it-by-Action (FIBA AI) is a zero-shot, edge-friendly system that retrieves action moments in raw video from a natural language query (for example, "cutting onion" or "opening a box").

## Team split
- Atul: query parsing, hand detection, object detection
- Tanishk: tracking, motion engine, action inferencing, segmentation
- Yash: pipeline integration, Flask backend, web UI
